# TFTStats

Personal Teamfight Tactics match statistics tracker and analytics platform.

## Overview

TFTStats crawls match history data from the Riot Games API to build a local 
database for analyzing meta trends, unit compositions, and player performance 
across patches.

## Purpose

This is a **personal research project** for self-improvement and data analysis. 
It is not a public-facing service and is not accessible by the community.

## Features

- Match history collection and storage
- Patch-based progress tracking
- Player performance analytics
- Meta trend analysis (unit compositions, trait popularity)

## Tech Stack

- **Language:** C# / .NET 10
- **Database:** PostgreSQL
- **Infrastructure:** Oracle Cloud (free tier)
- **API:** Riot Games TFT API
